# PRD — Instagram

## Goal

Use Instagram as the visual surface of the personal brand: carousels that teach, reels that demo accessibility patterns, and behind-the-build stories that make AI/accessibility approachable.

## Specifications

- ≥ 3 posts drafted per week, staged in `content/queue/instagram/YYYY-MM-DD-<slug>.md`.
- Each post: format = `carousel | reel | single | story`. Carousels include slide-by-slide outline (max 10 slides, each ≤ 90 chars headline + ≤ 220 chars body).
- Caption ≤ 2200 chars, hook in first line (truncation point at ~125 chars), CTA at end.
- Image prompts drafted for every visual slide via `image-prompt-writer`.
- Hashtag block: 8–15 hashtags, mixed reach (1 broad, ~5 mid, rest niche), bottom of caption or first comment.

## Forbidden actions

- Never auto-publish.
- Never reuse a carousel concept verbatim across platforms — adapt to medium.
- Never use stock-photo-feeling image prompts ("businessman pointing at chart"). Specific, contextual, on-brand.
- Never include claims that require sourcing without a research note ref.

## Voice

Visual-first. Captions support the visual, not the other way around. Slightly warmer than LinkedIn. Emoji sparingly, only where they aid scanning.
