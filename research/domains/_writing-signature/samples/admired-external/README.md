# Admired external posts

Reference material. These are LinkedIn posts the author admires for **hook
structure, pacing, argument shape, or tonal precision** — not voice samples.

**Do not copy phrasing.** The humanizer and writer agents read these only to
recognise hook patterns, pacing, and argument moves. Never reuse sentences.

**No emojis policy:** any post added here must be emoji-free. The original
emoji-heavy systematic-review post was deliberately removed because its style
conflicts with the author's voice.
